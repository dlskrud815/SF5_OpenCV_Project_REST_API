# Postman 테스트 결과
### 이미지 업로드 테스트:
메서드: POST
URL: http://localhost:5000/upload
Body: form-data 선택
Key: file
Type: File
Value: 업로드할 이미지 파일

### 이미지 조회 테스트:
메서드: GET
URL: http://localhost:5000/images/<image_id>
